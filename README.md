# GitScratch

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'git_scratch'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install git_scratch

## Usage

    $ rails g git_scratch:renew
